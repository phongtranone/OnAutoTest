# OnAutoTest
This Hybrid Automation Framework is engineered for robust functional testing across web applications and is designed for seamless integration within CI/CD pipelines in a DevOps environment.

- Prerequisite
- Installation

## Main Features
- The Cucumber Framework: enables the design of test cases using the Gherkin syntax, which facilitates behavior-driven development (BDD) through human-readable, structured language.
- Selenium WebDriver: supports cross-browser automation by interacting with multiple web browsers through a unified API.
